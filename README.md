# ScriLa

ScriLa is a scripting language with a Go and C++ like syntax that transpiles into Bash and PowerShell.

# Development
**Compile and run**  
`go run ./...`

**Build executable**  
`go build -o . ./...`

**Run all tests**
`go test -v ./...`
